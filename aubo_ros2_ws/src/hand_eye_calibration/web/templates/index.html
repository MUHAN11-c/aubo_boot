Eye-in-Hand角点法使用说明：
1. 先采集一张图像并提取角点（作为拍照姿态）
2. 然后移动机器人到不同位置，点击表格行捕获点选姿态
3. 每组数据使用相同的拍照姿态和不同的点选姿态
Eye-in-Hand姿态法使用说明：
1. 采集图像1并提取角点（获取初始标定板姿态）→ 捕获机器人姿态1
2. 移动机器人到新位置
3. 采集图像2并提取角点（获取结束标定板姿态）→ 捕获机器人姿态2
4. 重复上述步骤，至少采集2组运动数据
5. 点击"开始标定"进行姿态法标定

📖 自动标定流程

  1. 🔧 加载相机参数（自动从ROS2话题获取）
  2. ⚙️ 设置标定参数
    • 采样点数：建议8-15个
    • 工作空间：根据实际工作范围设置
    • Z轴高度：机器人末端高度
  3. 🚀 点击"开始自动标定"
  4. 🤖 系统自动执行：
    • 自动规划位姿（均匀分布）
    • 机器人移动→拍照→提取角点
    • 循环采集所有数据点
  5. ✅ 完成后自动执行标定计算
  6. 💾 保存标定结果
💡 优势：
  • 无需手动操作，全程自动化
  • 位姿分布均匀，提高标定精度
  • 节省时间，提高效率
  • 可重复性强，结果稳定

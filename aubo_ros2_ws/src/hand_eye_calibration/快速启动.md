# 手眼标定快速启动指南

## 🚀 方式一：使用快速启动脚本（推荐）

```bash
cd /home/mu/IVG/aubo_ros2_ws/src/hand_eye_calibration
./quick_start.sh
```

脚本会自动：
- ✅ 检测ROS2版本（humble/foxy/galactic）
- ✅ 加载ROS2环境
- ✅ 检查并编译工作空间（如需要）
- ✅ 启动手眼标定Web界面

启动后访问：**http://localhost:8080**

---

## 🚀 方式二：手动启动

### 1. 加载环境

```bash
cd /home/mu/IVG/aubo_ros2_ws
source /opt/ros/humble/setup.bash  # 根据实际ROS2版本调整
source install/setup.bash
```

### 2. 启动Web界面

```bash
ros2 launch hand_eye_calibration hand_eye_calibration_launch.py
```

### 3. 访问Web界面

打开浏览器访问：**http://localhost:8080**

---

## 🚀 方式三：命令行标定（无需Web界面）

如果已有标定数据，可以直接运行Python脚本：

```bash
cd /home/mu/IVG/aubo_ros2_ws/src/hand_eye_calibration/hand_eye_calibration_tool
python3 code/hand_eye_calibration.py
```

**注意**：此方式需要先准备好数据文件（见 `hand_eye_calibration_tool/README.md`）

---

## 📋 前置条件检查

启动前请确保：

1. ✅ **ROS2已安装**：`ros2 --version`
2. ✅ **工作空间已编译**：`colcon build --packages-select hand_eye_calibration`
3. ✅ **相机节点运行中**（如使用Web界面）：
   ```bash
   ros2 launch percipio_camera percipio_camera.launch.py
   ```
4. ✅ **机器人驱动运行中**（如使用Web界面）：
   ```bash
   ros2 launch demo_driver demo_driver.launch.py
   ```

## 🔧 常见问题

### 问题1：端口8080被占用

```bash
# 停止旧进程
pkill -f hand_eye_calibration_node

# 或使用其他端口
ros2 launch hand_eye_calibration hand_eye_calibration_launch.py web_port:=8081
```

### 问题2：找不到包

```bash
# 重新编译
cd /home/mu/IVG/aubo_ros2_ws
colcon build --packages-select hand_eye_calibration
source install/setup.bash
```

### 问题3：Web界面无法访问

- 检查防火墙设置
- 确认节点是否正常启动（查看终端输出）
- 尝试使用 `localhost` 或 `127.0.0.1` 而不是IP地址

#### 2.3 深度优先选择

```python
逻辑：
if depth_mm_actual > 0:
    z_to_use = depth_mm_actual  # 优先使用实际深度

✓ 边界处理正确：无效值（0）触发备用
```

#### 2.4 像素→相机3D坐标转换

```python
公式：
X = (u - cx) * Z / fx
Y = (v - cy) * Z / fy
Z = Z

示例（图像中心点）：
u=320.0, v=240.0, Z=500.0
X = (320.0 - 326.17) * 500.0 / 464.95 = -6.635 mm
Y = (240.0 - 240.68) * 500.0 / 465.01 = -0.731 mm
Z = 500.000 mm

验证：
✓ 公式正确
✓ 中心点接近(0, 0, Z)
✓ Z值保持不变
```

#### 2.5 相邻距离计算

```python
方法：
- 水平相邻：当前点到右侧点
- 垂直相邻：当前点到下侧点

数量：
水平距离数 = (cols-1) * rows = 9 * 7 = 63
垂直距离数 = cols * (rows-1) = 10 * 6 = 60
总距离数 = 123

验证：
✓ 计算数量正确：123个
✓ 距离公式正确：欧几里得距离
✓ 所有距离为正数
```

#### 2.6 对角线距离计算

```python
公式：
diagonal_theoretical = sqrt((cols-1)² + (rows-1)²) * square_size
                     = sqrt(9² + 6²) * 15.0
                     = sqrt(117) * 15.0
                     = 10.817 * 15.0
                     = 162.250 mm

实际计算：
diagonal_actual = norm(points_3d[0] - points_3d[-1])

验证：
✓ 理论公式正确
✓ 实际计算正确
✓ 第一个角点：左上角
✓ 最后一个角点：右下角
```

#### 2.7 误差统计

```python
计算项：
- 平均距离：mean(all_distances)
- 标准差：std(all_distances)
- 平均误差：abs(avg_distance - square_size)
- 误差百分比：(error / square_size) * 100

验证：
```

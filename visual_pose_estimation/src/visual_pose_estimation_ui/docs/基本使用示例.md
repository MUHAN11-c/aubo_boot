# 视觉姿态估计Web UI模块

## 基本使用示例

```bash
# 环境
source ~/.bashrc
cd ~/RVG_ws
source install/setup.bash

# 安装Python依赖
cd ~/RVG_ws/src/visual_pose_estimation_ui
pip3 install -r requirements.txt

# 启动Web UI服务
cd ~/RVG_ws/src/visual_pose_estimation_ui
./start_algorithm_server.sh

# 停止Web UI服务
./stop_web_services.sh
```

## 访问Web界面

启动服务后，在浏览器中访问：
- **Web UI**: http://localhost:8088/index.html
- **API服务**: http://localhost:8088/

## 测试示例

### Web界面操作

1. **工作流程选项卡**
   - 加载图像：点击"📂 加载图像"按钮
   - 采集图像：点击"📷 采集图像"按钮
   - 姿态估计：输入工件ID，点击"🎯 姿态估计"按钮
   - 目标抓取：点击"🤖 目标抓取"按钮
   - 打开/关闭夹爪：点击"✋ 打开夹爪"或"✊ 关闭夹爪"按钮

2. **模板设置选项卡**
   - 图像采集：输入工件ID和姿态ID，点击"📷 图像采集"
   - 获取姿态：选择姿态类型，点击"🤖 获取姿态"
   - 刷新模板：点击"🔄 刷新"按钮
   - 模板标准化：输入工件ID，点击"📐 模板标准化"
   - 删除模板：点击"🗑️ 删除模板"按钮

3. **Debug选项卡**
   - 加载/采集图像
   - 逐步处理图像
   - 保存调试结果

## 接口说明

### HTTP API接口

**GET /status** - 获取服务状态
- 返回服务运行状态和端口信息

**POST /api/capture_image** - 采集图像
```json
Request:
{
    "camera_id": "DA3234363"
}
Response:
{
    "success": true,
    "image_base64": "...",
    "image_path": "..."
}
```

**POST /api/estimate_pose** - 姿态估计
```json
Request:
{
    "input_image": "base64_encoded_image",
    "object_id": "211242785"
}
Response:
{
    "success": true,
    "success_num": 2,
    "confidence": [0.95, 0.88],
    "positions": [...],
    "grab_positions": [...],
    "preparation_positions": [...],
    "pose_images": [...],
    "vis_image": "..."
}
```

**POST /api/list_templates** - 获取模板列表
```json
Request:
{
    "templates_dir": ""
}
Response:
{
    "success": true,
    "count": 2,
    "templates": [
        {
            "template_id": "...",
            "workpiece_id": "...",
            "pose_id": "...",
            "image_path": "...",
            "image_base64": "..."
        }
    ]
}
```

**POST /api/get_robot_status** - 获取机器人状态
```json
Request: {}
Response:
{
    "success": true,
    "robot_status": {
        "is_online": true,
        "enable": true,
        "in_motion": false,
        "cartesian_position": {
            "position": {"x": 0.0, "y": 0.0, "z": 0.0},
            "orientation": {"x": 0.0, "y": 0.0, "z": 0.0, "w": 1.0},
            "euler_orientation_rpy_deg": [0.0, 0.0, 0.0]
        }
    }
}
```

**POST /api/write_plc_register** - 写入PLC寄存器
```json
Request:
{
    "address": "DB1.DBB3",
    "value": 0  // 0=打开夹爪, 1=关闭夹爪
}
Response:
{
    "success": true
}
```

**POST /api/capture_template_image** - 采集模板图像
```json
Request:
{
    "workpiece_id": "211242785",
    "pose_id": "1",
    "camera_id": "DA3234363"
}
Response:
{
    "success": true,
    "image_base64": "...",
    "image_path": "..."
}
```

**POST /api/save_template_pose** - 保存模板姿态
```json
Request:
{
    "workpiece_id": "211242785",
    "pose_id": "1",
    "pose_type": "camera_pose",
    "robot_status": {...}
}
Response:
{
    "success": true,
    "file_path": "..."
}
```

**POST /api/standardize_template** - 模板标准化
```json
Request:
{
    "workpiece_id": "211242785"
}
Response:
{
    "success": true,
    "processed_count": 5,
    "skipped_count": 0,
    "processed_pose_ids": ["1", "2", "3", "4", "5"]
}
```

**POST /api/visualize_grasp_pose** - 可视化抓取姿态
```json
Request:
{
    "template_image_path": "/path/to/template.jpg",
    "workpiece_id": "211242785",
    "pose_id": "1"
}
Response:
{
    "success": true,
    "image_base64": "..."
}
```

**POST /api/process_debug_step** - 处理调试步骤
```json
Request:
{
    "session_id": "debug_2024-01-01_12-00-00",
    "step_index": 0,
    "step_id": "original",
    "step_name": "读图并创建副本",
    "input_image": "base64_encoded_image",
    "is_redo": false
}
Response:
{
    "success": true,
    "result_image": "base64_encoded_image",
    "metadata": "{}"
}
```

## 启动流程

### 完整启动流程（推荐）

```bash
# 终端1: 启动ROS2核心节点（visual_pose_estimation）
cd ~/RVG_ws
source install/setup.bash
ros2 launch visual_pose_estimation visual_pose_estimation.launch.py

# 终端2: 启动Web UI服务
cd ~/RVG_ws/src/visual_pose_estimation_ui
source ~/RVG_ws/install/setup.bash
./start_algorithm_server.sh
```

### 访问Web界面

在浏览器中打开：http://localhost:8088/index.html

## 验证服务运行

```bash
# 检查服务是否运行
curl http://localhost:8088/status

# 检查端口占用
netstat -tlnp | grep 8088

# 检查进程
ps aux | grep algorithm_server.py
```

## 停止服务

```bash
# 方式1: 使用停止脚本
cd ~/RVG_ws/src/visual_pose_estimation_ui
./stop_web_services.sh

# 方式2: 手动停止
pkill -f algorithm_server.py

# 方式3: 在运行服务的终端中按 Ctrl+C
```

## 注意事项

1. **ROS2环境**: 确保ROS2环境已正确加载
2. **依赖包**: 确保 `interface` 包已编译
3. **端口占用**: 确保8088端口未被占用
4. **Python版本**: 建议使用Python 3.8+
5. **ROS2节点**: Web UI需要ROS2节点运行才能正常工作

## 故障排查

### 问题1: 服务无法启动

```bash
# 检查端口是否被占用
netstat -tlnp | grep 8088

# 检查Python依赖
pip3 list | grep rclpy

# 检查ROS2环境
source ~/RVG_ws/install/setup.bash
```

### 问题2: API调用失败

```bash
# 检查ROS2节点是否运行
ros2 node list | grep visual_pose_estimation

# 检查服务是否可用
ros2 service list | grep estimate_pose

# 查看服务日志
ros2 topic echo /visual_pose_estimation/status
```

### 问题3: 图像采集失败

```bash
# 检查相机服务是否运行
ros2 service list | grep software_trigger

# 检查图像话题
ros2 topic echo /camera_0/image_data
```
